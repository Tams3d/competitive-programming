# Competitive Programming

## Overview

This repository contains my **personal C++ solutions** to competitive programming problems.
- Only solution code is included.
- No problem statements, samples, or editorial content are stored.
- Each file is named using the **problem ID**, sometimes with a short identifier.

The repository is designed to be clean, compliant, and scalable.

## Platforms

- Codeforces (current)

Planned expansion:
- LeetCode
- CodeChef
- HackerRank
- ICPC style contests

Each platform will be organized into its own directory.

## Language and Style

- Language: **C++**
- Competitive programming oriented
- Fast I/O
- Focus on correctness, performance, and clarity
- Some problems may show progression from brute force to optimized solutions
